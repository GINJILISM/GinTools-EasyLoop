ストア提出に向けて必要な文案・確認項目・アイコン素材準備をまとめたものです。

主な機能:

- ムービー読み込み後すぐに編集画面へ

- MP4 書き出し
- GIF 書き出し
- 現在フレームのキャプチャ保存
- フォトライブラリ保存対応
- 小さな画面でも扱いやすいモバイル UI

## 4. App Icon 制作ガイド（Illustrator）

### 作成方針

- 元データはベクターで作る
- まず 1024 x 1024 の正方形をマスターにする
- 角丸は焼き込まず、基本は正方形で設計する
- 細い線や小さすぎる文字は避ける
- ひと目で「ループ」「切り出し」「軽やかさ」が伝わる構成にする

### Illustrator の作業手順

1. 新規ドキュメントを作成
2. アートボードを 1024 x 1024 px に設定
3. RGB カラーモードで作成
4. 背景 / シンボル / ハイライトをレイヤー分けする
5. 余白確認用に 820 x 820 px 程度の安全エリアガイドを置く
6. シンボルは中央寄せで構成する
7. 最後に 1024 px PNG を書き出す

### 推奨レイヤー構成

- background
- symbol-base
- symbol-accent
- highlight
- export-guides

### デザインの方向性案

#### 案 A: ループ矢印 + 切り出しフレーム

- 中央に柔らかい 2 本ループ矢印
- 外周にトリミングフレームを思わせる角表現
- 「編集してループにする」が一番伝わりやすい

#### 案 B: 1 コマ切り出し + リピート軌道

- 小さなフレーム矩形
- その周囲を回るループ軌道
- キャプチャ / クリップ切り出しの印象も出せる

#### 案 C: 再生三角 + 往復矢印

- 再生アイコンをベースにする
- 左右へ返るピンポン感をアクセントで入れる
- 動画アプリとしての認識が速い

### Illustrator での注意点

- 文字をアイコン本体に入れない
- 1 px 相当の細線は避ける
- 外周ぎりぎりまで要素を広げない
- 透明効果を多用しすぎない
- 書き出し前にアウトライン化の必要有無を確認する

## 5. リブランド時に触らないもの

以下は動作互換に関わるため、名称変更のついでに変えない。

- URL Scheme の `easyloop`
- Bundle Identifier の `com.ginjilism.gintool`
- App Group Identifier
- 共有シート連携の内部チャネル名

これらを変える場合は、共有シート / 既存インストール / 権限設定の再検証を別タスクとして扱うこと。
